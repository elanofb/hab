    Help - SoraMame.Block

    SoraMame.Block is block-type code editor for tiny script.

    How to use

    Block Tab

    • Move Block: Drag and Drop block.
    • Edit Expression: Click expression
    • Clear Trash-Can: Double-Click trash

    Code Tab: View code.

    Run Tab: Execute code.
